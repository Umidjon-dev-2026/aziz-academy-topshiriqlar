# 📚 Kutubxona boshqaruv tizimi (Library Management System)

## Loyiha haqida

Kutubxona boshqaruv tizimi — Python dasturlash tilida yozilgan oddiy konsol dasturi. Ushbu loyiha kutubxonadagi kitoblarni boshqarish uchun mo'ljallangan. Foydalanuvchi kitob qo'shishi, barcha kitoblarni ko'rishi, kitobni o'chirishi va dasturdan chiqishi mumkin.

## Imkoniyatlari

- 📖 Yangi kitob qo'shish
- 📚 Barcha kitoblarni ko'rish
- 🗑️ Kitobni o'chirish
- 🚪 Dasturdan chiqish

## Menyu

```
1. Kitob qo'shish
2. Kitoblarni ko'rish
3. Kitobni o'chirish
4. Chiqish
```

## Foydalanish

1. Dasturni ishga tushiring.
2. Menyudan kerakli bo'limni tanlang.
3. Buyruqlarni ketma-ket bajaring.
4. Chiqish uchun **4** ni tanlang.

## Texnologiyalar

- Python 3
- while sikli
- if / elif
- list
- input()
- print()
- for sikli
- enumerate()

## Loyiha maqsadi

Ushbu loyiha Python dasturlash tilining asosiy mavzularini amalda qo'llash uchun yaratildi. Dastur oddiy bo'lsa ham, ro'yxatlar (list), sikllar va shart operatorlari bilan ishlashni o'rganishga yordam beradi.

## Kelajakdagi rejalashtirilgan imkoniyatlar

- ✏️ Kitobni tahrirlash
- 💾 Ma'lumotlarni faylga saqlash
- 📊 Statistika bo'limi
- 👤 Foydalanuvchi tizimi
