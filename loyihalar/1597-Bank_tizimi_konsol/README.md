# Bank tizimi

## Loyiha haqida
Bu dastur oddiy bank tizimini simulyatsiya qiladi.

## Buyruqlar
- deposit <son> - balansga pul qo'shadi.
- withdraw <son> - balansdan pul yechadi.
- balance - joriy balansni ko'rsatadi.
- exit - dasturdan chiqadi.

## Ishlatish
Dasturni ishga tushiring va buyruqlarni ketma-ket kiriting.
